Etiquetas básicas de HTML

¿Qué es HTML?

Es un leguaje cifrado utilizado para el desarrollo de páginas web su siglas en ingles significan Hyper Text Markup Language y en español Lenguaje de marcas de hipertexto.

¿Qué es una etiqueta HTML?

Son fragmentos de texto entre los sigos mayor que (<) y menor que (>) con funciones y usos específicos; Se utilizan para escribir código HTML.

Tabla de Etiquetas básicas en HTML

Existe una serie de etiquetas que permiten la programación en el lenguaje HTML, estas son las más usadas:

ETIQUETAS HTML
Etiqueta Utilidad o Descripción
head Es la parte superior del documento HTML donde podremos indicar el título del documento.
body Indica la parte del cuerpo del contenido de un documento HTML.
div Elemento que es usado para agrupar otros elementos y actuar como plantilla de otros controles, nos ayuda a estructurar el documento en secciones.
a Esta etiqueta ayuda a crear un enlace a una página web. El atributo href, permite que se inserte el enlace a conectar. E atributo target, sirve para indicar si el enlace se abrirá en una nueva o en la misma ventana.
strong Sirve para dar énfasis a una parte en concreto del texto.
br Indica al navegador que viene un salto de línea para hacer el texto más leíble.
h1 …. h6 Sirven para insertar títulos dentro del contenido. Hay diferentes niveles de títulos, del 1 al 6. Las etiquetas h + número indican la importancia del título y del contenido.
img permite insertar y mostrar imágenes dentro del contenido.
ol – li – ul Las etiquetas OL y LI permiten crear listas, OL para listas ordenadas y UL para listas sin orden, dentro de las listas, los elementos se identifican con la etiqueta LI.
p Permite agrupar texto dentro de un parágrafo. El propósito es hacer el contenido más fácil de leer y mostrarlo de organizada.
span Permite personalizar el estilo de una palabra o texto.

Gauchat, J. D. (2012). El gran libro de HTML5, CSS3 y Javascript. Marcombo.
