Atributos

¿Que es un atributo?

Es un valor adicional que configura los elementos que ajusta su comportamiento de diversas formas para cumplir los criterios de los usuarios.

Atributos HTML básicos

Eston son algunos de los atributos mas usasos en la progrmacion de paginas web.

ATRIBUTOS HTML
Atributo Valor Descripcion
id nombre Establece un identificador a la etiqueta HTML.
class nombre Establece una clase (género) a una etiqueta HTML.
lang idioma Indica el idioma del contenido de la etiqueta HTML.
traslate yes / no Indica si el contenido de la etiqueta es traducible o no.
title título Mensaje mostrado en un tooltip (aviso amarillo emergente) al mover el ratón encima.
dta-* nombre Metadatos en la propia etiqueta.
accesskey atajo Combinación de teclas que puede pulsar el usuario para activar el elemento.
dir ltr / rtl Establece la direccionalidad del texto (left to right o right to left).
style estilos Aplica propiedades CSS directamente al elemento en cuestión.

Gauchat, J. D. (2012). El gran libro de HTML5, CSS3 y Javascript. Marcombo.

Mans (2021). Lenguaje HTML, https://lenguajehtml.com/html/introduccion/atributos-comunes-html/

Valores

Los valores son las propiedades que se pueden asignar a los elementos como nombre, idioma, estilo entre otros.
